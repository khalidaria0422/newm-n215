## Instructions

Create a responsive landing page using semantic HTML and CSS. Here is your [prototype you must re-create:](https://www.figma.com/design/EQNFo8Z7TuKb65qdGkR3qQ/N215-homework-1-prototype?node-id=0-1&t=wxuwtDk1ePSmtSql-1)

Your page should include the following sections:

- Header with navigation
- Hero/banner section
- Features or services section using cards or columns
- Footer

You must use Flexbox for layout structure.

Your layout should adjust reasonably for:

- desktop
- tablet
- mobile screen sizes

You may create your own theme, colors, and content, or use placeholder content.

## Requirements

### HTML

- Use semantic HTML elements when appropriate
- Organize content clearly
- Proper indentation and formatting

### CSS

Your project must demonstrate:

- Proper use of the CSS box model
- Margin and padding usage
- `box-sizing: border-box`
- Flexbox layout techniques
- Responsive sizing
- At least one media query

### Responsive

Your layout should:

- Avoid horizontal scrolling
- Resize cleanly on smaller screens
- Maintain readable spacing and structure
