## Requirements

### New Clients Page

Create a new page called: clients.html

This page must include:

- Header/navigation
- Hero or intro section
- Responsive client grid section
- Featured client/project section
- Footer

### Navigation

Your navigation must:

- Link between pages correctly
- Include working links for:
  - Home
  - Clients
- Maintain consistent styling across pages

### Responsive

Your layout should:

- Avoid horizontal scrolling
- Resize cleanly on smaller screens
- Maintain readable spacing and structure

### Client Grid

Your client section must:

- Use CSS Grid
- Include a minimum of 6 client/project cards
- Be responsive for desktop and mobile

Each card must include:

- Image
- Title
- Short description
- Button or link

### Responsive Design

Your project must include:

- At least 1 media query
- Responsive layout adjustments
- Proper spacing on desktop and mobile
- Readable typography on smaller screens

### CSS Grid

You must use:

```css
display: grid;
```

You should also include:

- grid-template-columns
- gap
- responsive adjustments

### Polish

Your project should include visual polish such as:

- Consistent spacing
- Rounded corners
- Hover effects
- Clean typography
- Shadows
- Consistent image sizing
- Professional alignment

### Image

All images should:

- Be resized/cropped consistently
- Match the design style
- Be stored in the images folder
- Use proper file names
